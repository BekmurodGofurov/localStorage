# localStorage Demo

This project is a small HTML page that demonstrates how to use browser `localStorage` to save a user's name.

## What it does

- Lets the user enter a name.
- Saves the name in `localStorage`.
- Shows the saved name when the page is opened again.
- Lets the user delete the saved name.

## How it works

- If no name is saved, the page asks the user to enter one.
- When a name is added, the input and add button are hidden and the saved name is shown.
- When the name is deleted, the page goes back to the initial state.

## Files

- `index.html` - contains the HTML, CSS, and JavaScript for the demo.
